# Yetis Ustam

Minimal Expo + React Native + TypeScript starter for the Yetis Ustam mobile game.

The app intentionally starts small:
- no backend
- no game logic yet
- no extra packages beyond the Expo blank TypeScript baseline
- a clean `src/` structure for branch-by-branch feature development

## Getting Started

```bash
npm install
npm run start
```

## Project Structure

```text
src/
  assets/
  components/
  game/
    data/
    engine/
    models/
  screens/
  store/
  utils/
```

`App.tsx` currently renders a single placeholder home screen so future features can be added incrementally.
